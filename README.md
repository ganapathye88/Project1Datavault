# dbt-data-vault-assignment

This repository contains a dbt project scaffolded for a Data Vault assignment.

## Structure

- `models/` - dbt models
- `seeds/` - dbt seed data
- `tests/` - dbt tests and custom tests
- `macros/` - dbt macros
- `snapshots/` - dbt snapshots
- `analyses/` - dbt analyses

## Setup

1. Install dbt.
2. Configure your profile in `~/.dbt/profiles.yml`.
3. Run `dbt debug`.
4. Run `dbt run`.
